Abrir o Site

Como hospedar no GitHub:

  1. Crie um repositório em github.com chamado visualstudio.com.br
  2. Conecte este diretório ao seu repositório:
  3. git remote add origin https://github.com/SEU-USUARIO/visualstudio.com.br.git
  4. Envie as mudanças: git push -u origin master
  5. Vá para Settings → Pages e selecione master branch
  6. Após alguns minutos, seu site estará em: SEU-USUARIO.github.io/visualstudio.com.br
