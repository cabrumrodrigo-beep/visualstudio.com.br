# Imagens do Visual Studio

Use esta pasta para armazenar todas as imagens do site:

- Logos
- Imagens de fundo
- Ilustrações
- Capturas de tela
- Avatares de equipe

## Como usar:

No HTML, use caminhos relativos:
```html
<img src="images/logo.png" alt="Logo Visual Studio">
<img src="../images/banner.jpg" alt="Banner do site">
```

## Recomendações:

- Use PNG para logotipos e gráficos com fundo transparente
- Use JPG para fotos e imagens complexas
- Use WebP quando possível (menor tamanho, mesma qualidade)
- Otimizar imagens antes de usar: [TinyPNG](https://tinypng.com/), [Squoosh](https://squoosh.app/)
